# NetRunner UI Ideas

## Атмосфера и Стилистика
- **Киберпанк интерфейс**: Использование моноширинных шрифтов, неоновых акцентов (зеленый/матрица или кибер-панк желтый).
- **Терминальный стиль**: Ответы агента могут сопровождаться префиксами `[SYSTEM]`, `[NEURAL_LINK]`, `>>>`.

## CLI Команды (Slash Commands)
Интерфейс должен поддерживать и визуализировать следующие команды:
- `/finish_quizz` — Принудительное завершение текущего квиза и переход к оценке. Должна быть кнопка в UI, которая отправляет эту команду.
- `/skip_question` — Пропуск текущего вопроса. Кнопка "Пропустить" в интерфейсе.

## Интерактивные элементы
- **Кнопки быстрого действия**: Вместо того чтобы только печатать, пользователь видит кнопки, которые эмулируют ввод слэш-команд.
- **Индикатор состояния**: Визуальное отображение того, что NetRunner находится в режиме "Протокола тестирования" (Quiz Mode).

## Особенности взаимодействия
- При вводе слэш-команд агент реагирует мгновенно, минуя долгие размышления LLM о намерениях.
- Возможность задать уточняющий вопрос прямо в процессе квиза, на что агент отвечает как умный помощник, а затем возвращает к контексту теста.
